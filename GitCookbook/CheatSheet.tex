\begin{homeworkProblem}
\begin{CJK*}{GBK}{song}
\section{Git Installation}

\subsection{Linux}


\begin{lstlisting}
>>git
The program 'git' is currently not installed. You can install it by typing:
sudo apt-get install git
\end{lstlisting}
输入git看看系统有没有安装Git，上述是没有安装Git的情况。

\subsubsection{Debian or Ubuntu Linux}
\begin{lstlisting}
>>sudo apt-get install git
\end{lstlisting}

\subsubsection{Other versions of Linux}
先下载源码，解压，再依次输入：
\begin{lstlisting}
>>./config
>>make
>>sudo make install
\end{lstlisting}

\subsection{Mac OS X}
安装homebrew，然后通过homebrew安装Git。参考homebrew文档：\url{http://brew.sh/}。\\
或者直接从AppStore安装Xcode，Xcode集成了Git，默认没有安装，需要运行Xcode，选择菜单“Xcode”，选择“Preferences”，在弹窗中找到“Downloads”，选择“Command Line Tools”，点“Install”就可以了。

\subsection{Windows}
msysgit: \url{https://git-for-windows.github.io}

\subsection{Configuration after installation}
\begin{lstlisting}
>>git config --global user.name <Name>
>>git config --global user.email <Email@sample.com>
\end{lstlisting}
由于Git是分布式版本控系统，每个系统必须给出Name和Email。由于--global参数，机器上的所有Git仓库都会使用这个配置。

\end{CJK*}

\end{homeworkProblem}
